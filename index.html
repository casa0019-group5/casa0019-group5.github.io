Noise Data Device

DATASET
This Data Device uses the real time water metre data from an individual tower of One Pool Street. This tower is split into two parts, SAC1 and SAC2. Both SAC1 and SAC2 have separate water metres that we collect data from.

DATA DEVICE (Physical)
 The data is visually represented using a color-coding system and a plain language description of various noise threshold. This approach aids users in understanding the impact of noise on their health.

DATA DEVICE (Digital)
 The Digital Twin of the data device provides additional information. It includes the sum of SAC1 and SAC2 for each individual floor in One Pool Street. In addition, it includes the total value for SAC1 and SAC2 separately in a barchart. Therefore, we can compare the difference in water usage between the separate sections of One Pool Street.

HOW TO
 Open the App and point it at the screen of the Data Device. The Augmented information will overlay with the Data Device, tap on the charts to change the visualisation from day to month. Use the dropdown menu to change space.
